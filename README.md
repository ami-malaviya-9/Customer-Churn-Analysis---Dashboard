# Customer-Churn-Analysis---Dashboard
An interactive Customer Churn Analysis Dashboard built using Power BI. The dashboard helps businesses understand customer behavior, identify churn patterns, and uncover key factors influencing customer retention.

## Project Overview

This project presents an interactive Customer Churn Analysis Dashboard built using Power BI. The dashboard helps businesses understand customer behavior, identify churn patterns, and uncover key factors influencing customer retention.

Customer churn is a critical metric for subscription-based and service businesses, as retaining existing customers is often more cost-effective than acquiring new ones.

This project demonstrates skills in data modeling, visualization, and business analytics using Power BI.

## Objectives

Analyze customer churn trends

Identify characteristics of customers likely to churn

Compare retained vs churned customers

Discover key drivers of churn

Provide actionable insights for improving retention strategies

## Dashboard Features
### Churn Overview

Total customers

Total churned customers

Churn rate (%)

Active vs churned distribution

### Customer Demographics Analysis

Churn by gender

Churn by age group or senior status

Churn by partner/dependent status

### Account & Contract Insights

Churn by contract type (Monthly / Yearly)

Churn by tenure range

Churn by payment method

Monthly charges comparison

### Service Usage Analysis

Churn by subscribed services

Impact of add-on services on retention

High-risk customer segments

### Financial Impact

Revenue loss from churned customers

Average revenue per user (ARPU)

High-value customers at risk
